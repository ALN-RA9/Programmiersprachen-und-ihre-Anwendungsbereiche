"HTML & CSS – Grundlagen des Webs"

Einführung in HTML & CSS

HTML (HyperText Markup Language) und CSS (Cascading Style Sheets) sind grundlegende Technologien zur Erstellung und Gestaltung von Webseiten. HTML strukturiert den Inhalt, während CSS für das Aussehen und Layout zuständig ist.

Verwendungsbereiche

HTML und CSS werden in vielen Bereichen des Webdesigns und der Webentwicklung eingesetzt:

  • Strukturierung von Webseiteninhalten (HTML)
  • Gestaltung von Layouts und Farben (CSS)
  • Responsives Design für mobile Geräte
  • Erstellung von Benutzeroberflächen für Webanwendungen

Beispielcode

Hier ist ein einfaches Beispiel für eine HTML-Datei mit eingebettetem CSS:


<!DOCTYPE html>
<html>
<head>
    <style>
        body {
            background-color: #f0f0f0;
            color: #333;
            font-family: Arial, sans-serif;
        }
    </style>
</head>
<body>
    <h1>Hello, World!</h1>
    <p>Dies ist eine einfache HTML-Seite mit CSS.</p>
</body>
</html>

Warum HTML & CSS lernen?

HTML und CSS bilden die Grundlage jeder Website. Sie sind einfach zu erlernen und der perfekte Einstieg in die Webentwicklung. Ohne HTML und CSS ist keine moderne Webseite denkbar.
