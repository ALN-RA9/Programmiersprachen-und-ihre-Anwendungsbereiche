Einführung in Objective-C

Objective-C ist eine objektorientierte Programmiersprache, die hauptsächlich für die Entwicklung von Software auf Apple-Plattformen wie macOS und iOS verwendet wurde. Sie basiert auf C und erweitert es um objektorientierte Konzepte.

Verwendungsbereiche

Objective-C wurde in folgenden Bereichen eingesetzt und wird teils weiterhin verwendet:

  • iOS- und macOS-App-Entwicklung (vor der Einführung von Swift)
  • Entwicklung mit dem Cocoa-Framework
  • Systemnahe Programmierung unter Apple-Betriebssystemen
  • Pflege älterer Apple-Anwendungen

Beispielcode

Hier ein einfaches Beispiel in Objective-C, das "Hello, World!" ausgibt:


#import <Foundation/Foundation.h>

int main(int argc, const char * argv[]) {
    @autoreleasepool {
        NSLog(@"Hello, World!");
    }
    return 0;
}

Warum Objective-C wählen?

Objective-C ist stark in bestehende Apple-Frameworks integriert und wird nach wie vor in vielen bestehenden Codebasen verwendet. Wer mit älteren Apple-Anwendungen arbeitet oder tiefere Einblicke in die Apple-Systemarchitektur gewinnen möchte, profitiert vom Verständnis dieser Sprache.
